# 🧩 노노그램 - 픽셀 퍼즐

토스 스타일의 깔끔한 노노그램 퍼즐 게임.

🎮 **[플레이하기](https://won-the-dev.github.io/toss-nonogram/)**

## 기능

- 🎯 50개 레벨 (5×5 ~ 15×15)
- 🎨 5개 프리셋 퍼즐 (하트, 별, 스마일, 집, 고양이)
- ✏️ 색칠/X표시 모드 전환
- ↩️ Undo/Redo
- 💡 힌트 시스템
- 📱 터치 & 드래그 입력
- ⏱️ 타이머
- 💾 자동 저장 (localStorage)
- 📱 모바일 우선 반응형

## 기술 스택

- React 19 + Vite
- Canvas API 렌더링
- GitHub Pages 배포

## 로컬 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
