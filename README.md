# 🧩 Pixel Puzzle — 노노그램

모바일 최적화된 노노그램 퍼즐 게임

## 특징

- 🎯 150개 핸드메이드 퍼즐 (5×5 ~ 15×15)
- 🎮 터치 모드 + 컨트롤러 모드 (D-pad)
- 🌙 다크 모드
- 💾 자동 저장 (localStorage)
- 🖼️ 컬렉션 시스템 — 퍼즐을 모아 큰 그림 완성
- 🔊 Web Audio API 사운드 효과
- 📳 햅틱 피드백

## 플레이

👉 [https://won-the-dev.github.io/toss-nonogram/](https://won-the-dev.github.io/toss-nonogram/)

## 기술 스택

- React 19 + Vite
- Canvas API (게임 보드 렌더링)
- Web Audio API (오실레이터 기반 사운드)
- GitHub Pages 배포

## 로컬 실행

```bash
npm install
npm run dev
```

## 빌드 & 배포

```bash
npm run build
npx gh-pages -d dist
```
