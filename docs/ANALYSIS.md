# 노노그램 앱 경쟁사 분석

*분석일: 2026-02-06*

## 1. 주요 경쟁 앱

### Nonogram.com (Easybrain)
- **다운로드:** 10M+
- **평점:** 4.7
- **강점:** 깔끔한 UI, 일일 챌린지, 시즌 이벤트
- **BM:** 광고 + 프리미엄 구독

### Nonograms Katana
- **다운로드:** 5M+
- **평점:** 4.8
- **강점:** RPG 요소, 유저 퍼즐 공유, 50x50 지원
- **BM:** 광고 + VIP 구매

### Picross Luna
- **특징:** 스토리 모드, 캐릭터
- **BM:** 챕터별 구매

## 3. 수익 모델 (BM)

### 광고
| 타입 | 시점 | 예상 eCPM |
|-----|------|----------|
| 전면 광고 | 레벨 클리어 후 | $5-15 |
| 리워드 광고 | 힌트 획득 | $10-30 |
| 배너 | 하단 고정 | $1-3 |

### 인앱 결제
| 아이템 | 가격 | 전환율 (예상) |
|-------|------|-------------|
| 광고 제거 | ₩9,900 | 2-5% |
| 힌트 10개 | ₩1,100 | 5-10% |
| 힌트 50개 | ₩4,400 | 3-5% |
| VIP (무제한) | ₩19,900 | 1-2% |

- 토스 IAA (광고)
- 토스페이 (결제)
- 현재 수수료 없음 → 마진 극대화

## 4. 콘텐츠 전략

### 그림 카테고리
1. **동물** (40%) - 고양이, 강아지, 토끼
2. **음식** (20%) - 과일, 디저트
3. **일상** (20%) - 하트, 별, 집
4. **시즌** (20%) - 크리스마스, 할로윈

## 5. 핵심 인사이트

### 성공 요인
1. **중독성:** 5분 안에 완료 가능한 퍼즐
2. **성취감:** 그림 완성의 만족감
3. **진입장벽 낮음:** 규칙 단순
4. **무한 콘텐츠:** 자동 생성 가능

### 우리 전략
1. **MVP 빠르게:** 5x5 ~ 10x10 먼저
2. **한국 감성:** 토스/한국 테마로 차별화
3. **수익화:** 리워드 광고 + 힌트 판매
4. **리텐션:** 일일 챌린지
